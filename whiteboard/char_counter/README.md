## Problem

write a fn() that takes in 1st argument as a string and returns a letter count of each character.

### Running

Examples

[x] example counter 
```
const newCounter = returnCharCnt(charCounter.test.strings.four);
console.log(newCounter); // { 1: 1, ' ': 45, '.': 1 }
```
